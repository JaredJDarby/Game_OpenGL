Application can be started by running the executable file in the solution directory.

Controls:
W/Up Arrow/Num 8    - increase pitch.
S/Down Arrow/Num 2  - decrease pitch.
A/Left Arrow/Num 4  - decrease yaw.
D/Right Arrow/Num 6 - increase yaw.
Q/Num 7             - decrease roll.
E/Num 9             - increase roll.
Esc                 - exit program.
